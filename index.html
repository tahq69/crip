About CRIP

CRIP environment is a collection of software tools for inspiring our community of developers to push the boundaries of web. Crip is gateway through with you can discover the tools you need, whether you’re building high-performance apps, front-end or back-end API.

You can find latest information about interesting stuff on public link blog created by out team. You can start post your stuff, simply register in blog and download extension for browser.

Components

CRIP environment is fully open source and available on GitHub for Laravel, AngularJs, Node.js and design set of reusable components. Modular structure allows use each component separate or use all peace's together to create your own large application.
